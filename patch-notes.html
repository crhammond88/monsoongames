V 1.1.0

                General
        • Saves added
          • Settings and gameplay data are now saved
          • Time Trackers added
            • Best completion times are now tracked and displayed
          • Gating added
            • Necromancers, difficulties and potential opponents must now be unlocked
        • Sound Effects added
        • Gameplay
          • Basic attacks will now always complete if not canceled by the player
          • Necromancer respawn timers adjusted to provide a bonus for kill leads
            • From [Level + 6] to [(Deaths - Kills)/2 + Level + 5]
              • Negative (Deaths - Kills) does not reduce the timer
          • The aggression range for auto attacks is now determined by Attack Range
          • Minion Attack Damage growth increased
            • Melee from 4 to 5
            • Ranged from 3 to 4
          • Enemy AI improved
        • HUD
          • Pause Menu now displays Challenge information
        • Misc
          • Border added to bottom of the battlefield
          • Special effect added for Necromancer level up
          • Title sequence sped up slightly
          • Loading times for slow connections reduced significantly
        • Bug Fixes
          • Skeleton dance animations now loop
          • Minions no longer prioritize Necromancers over minions
          • Auto-attack now disables while teleporting
          • Respawn timers now display correctly on the game clock
          • Fixed an issue that occasionally caused Necromancers to slide instead of run
          • Fixed an issue with Wasserbar’s Runts failing to seek a target
          • Reverse Gravity no longer dips units underground
